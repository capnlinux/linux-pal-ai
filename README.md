
# 🐧 LinuxPal — AI Linux Assistant for Everyone

> No long videos. No confusing man pages. Just ask and get help at your level.

---

## What is LinuxPal?

LinuxPal is a free, open-source AI assistant built specifically for Linux users.

Whether you just installed Linux for the first time, or you've been using it for years — LinuxPal explains things **at your level**, not above it.

---

## Why I built this

I started learning Linux in 2015. It was hard. The tutorials were long, the errors were confusing, and nobody explained things simply.

After 10 years, Linux became easy for me — but I noticed new users still face the same struggles.

So I built LinuxPal. Not to take credit. Just to help.

This project belongs to the community. Use it, improve it, share it.

---

## Features

- 🌱 **5 skill levels** — Amateur, Beginner, Semi-Pro, Pro, Expert
- 🤖 **AI-powered answers** — real explanations, not copy-pasted man pages
- ⚡ **Fast** — no 20-minute videos, just ask and get the answer
- 🌍 **For everyone** — students, hobbyists, professionals
- 💸 **Completely free** — no account, no payment, no catch

### Skill Levels Explained

| Level | Who it's for | How it answers |
|-------|-------------|----------------|
| 🌱 Amateur | Never used a terminal | Plain English, everyday analogies |
| 📘 Beginner | Knows a few basic commands | Simple words + commands explained |
| ⚙️ Semi-Pro | Uses Linux regularly | Commands with context and the "why" |
| 💻 Pro | Experienced Linux user | Direct, technical, no hand-holding |
| 🔥 Expert | Sysadmin / DevOps / Kernel level | Raw, terse, internals included |

---

## How to Use

1. Open the app (link below)
2. Pick your skill level
3. Ask anything — errors, commands, concepts, file system, networking, anything Linux

**Example questions:**
- *"permission denied when I run my script"*
- *"what does chmod 755 mean"*
- *"my disk is full, what do I do"*
- *"difference between apt and apt-get"*
- *"how do I check which process is using port 3000"*

## Tech Stack
